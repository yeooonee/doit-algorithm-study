# 정렬 알고리즘

## 정렬이란?

이름, 학번, 학점 등의 키를 항목값의 대소 관계에 따라 데이터 집합을 일정한 순서로 바꾸어 늘어놓는 작업

- **오름차순 정렬**, **내림차순 정렬**
- 데이터를 **교환**, **선택**, **삽입**하면서 정렬을 완료함

| 구분 | 설명 |
|------|------|
| **안정적인 알고리즘** | 같은 값의 원소 순서가 정렬 후에도 유지됨 |
| **불안정적인 알고리즘** | 같은 값의 원소 순서가 무작위로 바뀔 수 있음 |
| **내부 정렬** | 정렬할 모든 데이터를 하나의 배열에 저장할 수 있는 경우 사용 |
| **외부 정렬** | 데이터가 많아 하나의 배열에 저장할 수 없는 경우 사용 |

<br/>

## 버블 정렬 (단순 교환 정렬)

이웃한 두 원소의 대소 관계를 비교하여 필요에 따라 교환을 반복하는 알고리즘
- 모든 정렬이 끝나려면 패스를 **N-1번** 수행해야 함
- 비교 횟수: **n(n-1)/2**

### 기본
```python
from typing import MutableSequence

def bubble_sort(a: MutableSequence) -> None:
    n = len(a)
    for i in range(n - 1):
        for j in range(n - 1, i, -1):
            if a[j - 1] > a[j]:
                a[j - 1], a[j] = a[j], a[j - 1]
```

### 개선 1 - 교환 발생 여부 체크

교환이 한 번도 일어나지 않으면 이미 정렬된 것이므로 조기 종료

```python
from typing import MutableSequence

def bubble_sort(a: MutableSequence) -> None:
    n = len(a)
    for i in range(n - 1):
        exchng = 0
        for j in range(n - 1, i, -1):
            if a[j - 1] > a[j]:
                a[j - 1], a[j] = a[j], a[j - 1]
                exchng += 1
        if exchng == 0:
            break
```

### 개선 2 - 마지막 교환 위치 추적

마지막으로 교환이 일어난 위치까지만 다음 패스 범위로 설정

```python
from typing import MutableSequence

def bubble_sort(a: MutableSequence) -> None:
    n = len(a)
    k = 0
    while k < n - 1:
        last = n - 1
        for j in range(n - 1, k, -1):
            if a[j - 1] > a[j]:
                a[j - 1], a[j] = a[j], a[j - 1]
                last = j
        k = last
```

### 세이커 정렬 (양방향 버블 정렬 / 칵테일 정렬)

버블 정렬의 개선 알고리즘. 홀수 패스에서는 가장 작은 원소를 맨 앞으로, 짝수 패스에서는 가장 큰 원소를 맨 뒤로 이동시킴 <br/>
예를 들어,  `9, 1, 3, 4, 6, 7, 8`에서 9가 맨 앞에 있으면 단순 버블 정렬은 비효율적임 <br/>
세이커 정렬은 *양방향* 으로 스캔해 빠르게 처리함

```python
from typing import MutableSequence

def shaker_sort(a: MutableSequence) -> None:
    left = 0
    right = len(a) - 1
    last = right
    while left < right:
        for j in range(right, left, -1):
            if a[j - 1] > a[j]:
                a[j - 1], a[j] = a[j], a[j - 1]
                last = j
        left = last

        for j in range(left, right):
            if a[j] > a[j + 1]:
                a[j], a[j + 1] = a[j + 1], a[j]
                last = j
        right = last
```

## 단순 선택 정렬

가장 작은 원소부터 선택해 알맞은 위치로 옮기는 작업을 반복하며 정렬하는 알고리즘
- 비교 횟수: **(n² - n) / 2**

```python
from typing import MutableSequence

def selection_sort(a: MutableSequence) -> None:
    n = len(a)
    for i in range(n - 1):
        min = i
        for j in range(i + 1, n):
            if a[j] < a[min]:
                min = j
        a[i], a[min] = a[min], a[i]
```

## 단순 삽입 정렬

주목한 원소보다 더 앞쪽의 알맞은 위치에 삽입하며 정렬하는 알고리즘

```python
from typing import MutableSequence

def insertion_sort(a: MutableSequence) -> None:
    n = len(a)
    for i in range(1, n):
        j = i
        tmp = a[i]
        while j > 0 and a[j - 1] > tmp:
            a[j] = a[j - 1]
            j -= 1
        a[j] = tmp
```

## 시간 복잡도 정리

| 알고리즘 | 시간 복잡도 | 비고 |
|----------|-------------|------|
| 버블 정렬 | O(n²) | |
| 단순 선택 정렬 | O(n²) | |
| 단순 삽입 정렬 | O(n²) | |
| **이진 삽입 정렬** | **O(n²)** | 비교 횟수는 O(n log n)으로 개선 |

### 이진 삽입 정렬

이진 탐색을 활용해 삽입 위치를 찾는 개선된 삽입 정렬.
- **비교 횟수**: O(n log n) 
- **이동 횟수**: O(n²) 
- **전체 시간 복잡도**: O(n²)
- bisect 모듈의 `insort()` 함수를 제공

```python
from typing import Mutable Sequence 

def insertion_sort(a: MutableSequence) -> None:
	n = len(a)
	for i in range(1, n):
		j = i
		tmp = a[i]
		while j > 0 and a[j - 1] > tmp:
			a[j] = a[j - 1]
			j -= 1
		a[j] = tmp
```
