# 재귀 알고리즘
## 재귀
- 자기 자신을 포함하고 다시 자기 자신을 사용하여 정의되는 경우
- 프로그램을 간결하고 효율성 좋게 작성 가능

### 팩토리얼 
- 양의 정수를 순서대로 곱함.
- 10! = 10 * 9! 로 factorial을 재귀적으로 호출

## 직접 재귀
- 자신과 똑같은 함수를 호출하는 방식

## 간접 재귀
- 두개의 함수가 번갈아가며 서로를 호출 

## 유클리드 호제법
- 최대 공약수를 재귀적으로 구하는 방법
- 두 정숫값이 주어질 때 큰 값 / 작은 값, 나누어 떨어지면 작은 값이 최대 공약수가 됨. 

## 순수한 재귀
- 재귀 호출을 한 함수내에서 여러번 실행하는 함수

## 하향식 분석
- 상위에서 함수를 호출하며 분석 
- 함수 호출부터 계단식으로 자세히 조사

## 상향식 분석
- 아래쪽부터 쌓아 올리며 분석
- 가장 작은 값으로 함수 호출하여 처리 확인
