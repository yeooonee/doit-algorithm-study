# 파이썬 
## 예외처리
- try 문을 이용하여 예외 처리
```
try:
    # 실행할 코드 (오류가 날 수 있는 부분)
except 예외종류:
    # 오류가 발생했을 때 실행할 코드
```

### try-except & try-finally 차이 ⭐️
- try-except 에서는 예외를 잡지 못하면 프로그램이 종료됨
- tyr-finally 에서는 오류가 나도 반드시 정리해야 할 작업이 있을 때 사용

### Raise 문을 이용
- Raise 문으로 의도적으로 예외처리 가능
