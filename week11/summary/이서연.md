# 문자열 검색
## 브루트 포스법
- 어떤 문자열 안에 다른 문자열이 포함되어 있는지 검사
- 텍스트 : 검색되는 쪽의 문자열
- 패턴 : 찾아내는 문자열
- 이미 검사한 위치를 기억하지 못하므로 브루트 포스법은 효율이 좋지 않음.
    - 0,1,2 를 검사하였고 2에서 일치하지 않음 > 1,2,3 도 일치하지 않을 것 예측 가능. 
    - 근데 브루트 포스는 걸러내지 않고 전부 다 검사.

## KMP
- 검사했던 결과 재활용
- 텍스트와 패턴 안에서 겹치는 문자열을 찾아내 검사를 다시 시작할 위치 저장
- 패턴에서 겹치는 문자열 찾아냄 > 현재 위치에서 패턴이 일치하지 않을 경우 다음 패턴은 겹치는 문자열에서 시작
- 건너뛰기 표 : 패턴 저장한 표 

## 보이어 무어
- 패턴의 끝 문자에서 시작하여 앞쪽을 향해 검사 수행
- 패턴이 실패했을 때, 불일치한 문자 기준으로 일치하는 부분으로 이동


```
보이어 무어가 가장 크게 이동하며 제일 효율적임.
```
