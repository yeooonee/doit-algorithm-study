## 하노이의 탑

### 개념
- 작은 원반은 큰 원반 위에만 놓을 수 있다.
- 기둥 3개를 이용해 원반을 모두 옮기는 문제
- 재귀 알고리즘의 대표 예제

<br/><br/>

## 8퀸 문제

### 개념
8×8 체스판에 8개의 퀸을 서로 공격하지 못하도록 배치하는 문제

퀸의 공격 범위
- 가로
- 세로
- 대각선

총 **92가지** 해결 방법이 있다.

<br/><br/>

## 경우의 수 줄이기

### 단순 배치

`64 × 63 × ... × 57 = 178조 이상`

### 규칙 적용

1. 각 행에 퀸 1개
2. 각 열에 퀸 1개

`8⁸ = 16,777,216` 으로 감소

<br/><br/>

### 분기 작업 (Branching)

가능한 모든 경우를 가지처럼 뻗어가며 탐색하는 방법

예)

```text
1행
├─ 0열
├─ 1열
├─ 2열
└─ ...
```
### 분할 정복법(분할 해결법)

하노이의 탑이나 8퀸 문제처럼 큰 문제를 작은 문제로 분할하고, 작은 문제 풀이법을 결합하여 전체 풀이법을 얻는 방법

### 한정 작업 (Bounding)

조건에 맞지 않는 경우는 더 이상 탐색하지 않고 제거<br/>
(퀸을 예로 들면 같은 열, 같은 대각선에 퀸이 있으면 중단)

### 분기 한정법 (Branch and Bound)

분기 + 한정을 함께 사용하는 방법

<br/><br/>

## 8퀸 해결

```python
pos = [0] * 8
```
각 행의 퀸 위치 저장
<br/><br/>

```python
flag_a = [False] * 8
```
열 검사
<br/><br/>

```python
flag_b = [False] * 15
```
↘ 대각선 검사 (`i + j`)
<br/><br/>

```python
flag_c = [False] * 15
```
↙ 대각선 검사 (`i - j + 7`)
<br/><br/>

### 배치 조건

```python
if (not flag_a[j]
    and not flag_b[i+j]
    and not flag_c[i-j+7]):
```
`같은 열 X` & `↘ 대각선 X` & `↙ 대각선 X`일 때만 퀸 배치 가능
