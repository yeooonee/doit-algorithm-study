# 리스트(List)

**리스트**: 데이터에 순서를 매겨 늘어놓은 자료구조
리스트 구조에는 크게 **선형 리스트**와 **연결 리스트**가 있다.
- 선형 리스트: 데이터를 메모리에 빈틈없이 순서대로 연속해서 저장하는 구조 (예: 배열).
- 연결 리스트: 데이터들을 메모리 상에 흩어놓고 각 데이터가 다음 데이터의 위치(주소)를 가리키며 연결하는 구조.

## 연결 리스트 (Linked List)

- **노드(node)**: 리스트를 구성하는 각각의 원소
- 노드는 **데이터**와, 뒤쪽 노드를 가리키는(참조하는) **포인터**를 가진다.

### 노드의 명칭

| 용어 | 의미 |
|---|---|
| 머리 노드 (head node) | 맨 앞에 있는 노드 |
| 꼬리 노드 (tail node) | 맨 끝에 있는 노드 |
| 앞쪽 노드 (predecessor node) | 각 노드의 바로 앞 노드 |
| 뒤쪽 노드 (successor node) | 각 노드의 바로 뒤 노드 |

## 포인터를 이용한 연결 리스트

### Node 클래스

`Node`는 데이터용 필드 `data`와, 자기 자신과 같은 클래스형의 인스턴스를 참조하는 필드 `next`를 갖는다.

> 이처럼 **자기 자신과 같은 형의 인스턴스를 참조하는 필드**가 있는 구조를 **자기 참조형**이라고 한다.

**필드 구성**
- `data` : 데이터
- `next` : 뒤쪽 포인터

```python
from __future__ import annotations
from typing import Any, Type

class Node:
    """연결 리스트용 노드 클래스"""

    def __init__(self, data: Any = None, next: Node = None):
        """초기화"""
        self.data = data    # 데이터
        self.next = next    # 뒤쪽 포인터
```

- `__init__()` : 전달받은 `data`와 `next`를 각 필드에 대입

<br/><br/>

### LinkedList 클래스 (연결 리스트 구현)


| 메서드 | 기능 |
|---|---|
| `search` | 값으로 노드 검색 |
| `add_first` / `add_last` | 머리 / 꼬리에 노드 삽입 |
| `remove_first` / `remove_last` | 머리 / 꼬리 노드 삭제 |
| `remove` | 임의의 노드 삭제 |
| `clear` | 전체 노드 삭제 |
| `next` | 주목 노드를 다음으로 이동 |
| `__iter__` / `__next__` | 반복(iteration) 지원 |

<summary>LinkedList 클래스 소스 정리</summary>

```python
# 초기화 및 노드 개수 반환
def __init__(self) -> None:
    self.no = 0          # 노드의 개수
    self.head = None     # 머리 노드
    self.current = None  # 주목 노드

# 연결 리스트의 노드 개수를 반환
def __len__(self) -> int:
    return self.no

# 검색 수행 (data와 값이 같은 노드를 검색)
def search(self, data: Any) -> int:
    cnt = 0
    ptr = self.head
    while ptr is not None:
        if ptr.data == data:
            self.current = ptr
            return cnt
        cnt += 1
        ptr = ptr.next
    return -1

# 연결 리스트에 data가 포함되어 있는지 확인
def __contains__(self, data: Any) -> bool:
    return self.search(data) >= 0

# 머리 노드 삽입
def add_first(self, data: Any) -> None:
    ptr = self.head  # 삽입하기 전의 머리 노드
    self.head = self.current = Node(data, ptr)
    # Node(data, ptr): data = 새로 넣을 값, ptr = 이전 머리 노드(연결을 위해 참조)
    self.no += 1

# 꼬리 노드 삽입
def add_last(self, data: Any):
    if self.head is None:     # 리스트가 비어 있으면
        self.add_first(data)  # 맨 앞에 노드를 삽입
    else:
        ptr = self.head
        while ptr.next is not None:
            # 한 번에 끝으로 갈 수 없으므로 한 칸씩 이동하며 마지막 노드를 찾는다
            ptr = ptr.next
        ptr.next = self.current = Node(data, None)
        self.no += 1

# 머리 노드 삭제
def remove_first(self) -> None:
    if self.head is not None:  # 리스트가 비어 있지 않으면
        self.head = self.current = self.head.next
        self.no -= 1

# 꼬리 노드 삭제
def remove_last(self):
    if self.head is not None:
        if self.head.next is None:  # 노드가 1개 뿐이라면
            self.remove_first()     # 머리 노드를 삭제
        else:
            ptr = self.head         # 스캔 중인 노드
            pre = self.head         # 스캔 중인 노드의 앞쪽 노드

            while ptr.next is not None:
                pre = ptr
                ptr = ptr.next
            pre.next = None         # pre는 삭제 후 꼬리 노드가 됨
            self.current = pre
            self.no -= 1

# 임의의 노드 삭제
def remove(self, p: Node) -> None:
    if self.head is not None:
        if p is self.head:        # p가 머리 노드이면
            self.remove_first()   # 머리 노드를 삭제
        else:
            ptr = self.head

            while ptr.next is not p:
                ptr = ptr.next
                if ptr is None:
                    return        # p가 리스트에 존재하지 않으므로 종료
            ptr.next = p.next
            self.current = ptr
            self.no -= 1

# 주목 노드를 삭제
def remove_current_node(self) -> None:
    self.remove(self.current)

# 전체 노드를 삭제
def clear(self) -> None:
    while self.head is not None:  # 전체가 빌 때까지
        self.remove_first()       # 머리 노드를 삭제
    self.current = None
    self.no = 0

# 주목 노드를 한 칸 뒤로 이동
def next(self) -> bool:
    if self.current is None or self.current.next is None:
        return False              # 이동할 수 없음
    self.current = self.current.next
    return True

# 주목 노드를 출력
def print_current_node(self) -> None:
    if self.current is None:
        print('주목 노드가 존재하지 않습니다.')
    else:
        print(self.current.data)

# 모든 노드를 출력
def print(self) -> None:
    ptr = self.head
    while ptr is not None:
        print(ptr.data)
        ptr = ptr.next

# 이터레이터 반환
def __iter__(self) -> LinkedListIterator:
    return LinkedListIterator(self.head)
```

### LinkedListIterator 클래스

```python
class LinkedListIterator:

    def __init__(self, head: Node):
        self.current = head

    def __iter__(self) -> LinkedListIterator:
        return self

    def __next__(self) -> Any:
        if self.current is None:
            raise StopIteration
        else:
            data = self.current.data
            self.current = self.current.next
            return data
```

## 이터러블 객체와 이터레이터의 구현

### 이터러블(Iterable) vs 이터레이터(Iterator)

- **이터러블**: `str`형 문자열, `list`형 리스트, `tuple`형 튜플 등 **반복 가능(iterable)** 한 객체.
- 원소를 1개씩 꺼내는 구조를 가진다.
    - 이터러블 객체를 내장 함수 `iter()`에 전달하면 → 그 객체에 대한 **이터레이터(iterator)** 를 반환한다.
- **이터레이터**: 데이터가 줄지어 늘어선 것을 표현하는 객체
    - `__next__()` 함수를 호출하거나 내장 함수 `next()`에 이터레이터를 전달하면, 줄지어 늘어선 원소를 순차적으로 꺼낸다.
    - 꺼낼 원소가 없으면 `StopIteration` 예외를 발생시킨다.
    - `next()`를 처음 호출하면 맨 앞 원소, 2번째 호출하면 2번째 원소... 호출할 때마다 다음 원소를 꺼내는 방식

### `LinkedList`와의 연결

`LinkedList`는 **이터러블**이 되도록 이터레이터를 구현한다.
이터레이터를 나타내는 것이 바로 `LinkedListIterator` 클래스이며, 다음 두 가지를 구현해야 함.

1. `__iter__()` — `__next__()` 함수를 갖는 이터레이터를 반환
2. `__next__()` — 다음 원소를 반환하고, 반환할 원소가 없으면 `StopIteration` 예외를 발생
